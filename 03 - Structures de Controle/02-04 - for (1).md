# `for` (1)

Que va afficher à l'exécution chacun des groupes d'instructions ci-dessous ?

~~~cpp
// 1
for (int i = 1; i < 10; ++i) {
   cout << i << " ";
}
~~~


<details>
<summary>Solution</summary>

~~~
1 2 3 4 5 6 7 8 9
~~~
</details>


~~~cpp
// 2
for (int i = 1; i < 10; i += 2) {
   cout << i << " ";
}
~~~

<details>
<summary>Solution</summary>

~~~
1 3 5 7 9
~~~
</details>

~~~cpp
// 3
for (int i = 10; i > 1; --i) {
   cout << i << " ";
}
~~~

<details>
<summary>Solution</summary>

~~~

~~~
</details>
